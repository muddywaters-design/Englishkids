📋 פעילות אחרונה

☁️ סנכרון ל-Google Sheets

כל סשן (גם נטוש) יישלח אוטומטית לגיליון בחשבון הגוגל שלך.

📖 הוראות התקנה
הגדרה חד-פעמית (5 דקות):
  1. פתח sheets.new ליצירת גיליון חדש
  2. תפריט: Extensions → Apps Script
  3. מחק את הקוד הקיים והדבק את הקוד למטה
  4. שמור (Ctrl/Cmd+S)
  5. לחץ Deploy → New deployment
  6. בחר Type: Web app
  7. Execute as: Me, Who has access: Anyone
  8. לחץ Deploy ואשר הרשאות
  9. העתק את ה-Web app URL והדבק כאן למעלה

📤 ייצוא ושיתוף
